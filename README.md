# Snakes & Ladders - LLD Practice 🎲

A **Low-Level Design (LLD) practice project** implementing the classic board game **Snakes & Ladders** in Java. Focused on practicing **OOP, class modeling, and design patterns**.

---

## 📌 Summary

This project demonstrates how to model a real-world game using object-oriented principles:

* Proper class design
* Encapsulation of game logic
* Interaction between objects (Player, Board, Dice, Snake, Ladder)

It is intended as a **practice exercise for system modeling and design interviews**.

---

## 🧠 Key Topics

* Object-Oriented Design (Classes, Objects, Inheritance)
* Encapsulation & Abstraction
* Game logic implementation
* Low-Level Design patterns (Strategy, Singleton, etc.)
* Unit testable design

---

## 📂 Repository Structure

```text id="snlstr1"
src/
 ├── model/       # Core classes: Player, Board, Snake, Ladder, Dice
 ├── service/     # Game engine / logic
 ├── utils/       # Utility classes
 └── Main.java    # Entry point to start the game
```

---

## ▶️ How to Run

1. Clone the repo:

```bash id="snlrun1"
git clone https://github.com/KavitaJadhav/snakes_ladders.git
cd snakes_ladders
```

2. Compile and run:

```bash id="snlrun2"
javac src/*.java
java src.Main
```

3. Play the game from console input
